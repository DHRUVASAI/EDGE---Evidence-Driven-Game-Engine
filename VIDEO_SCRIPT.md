# CricMetrics - 1 Minute Demo Script 🎬
**Duration:** 60 seconds  
**Pace:** Fast-paced, energetic, technical

---

## 🎯 Script (Exactly 60 seconds - 150 words)

**[0-10s] Hook + Overview**
> "Meet CricMetrics - an AI-powered cricket analytics platform built with Next.js 16 and TypeScript, processing over 3 million ball-by-ball deliveries across 22,000 matches stored in PostgreSQL."

**[10-20s] Real-Time Intelligence**
> "Live match tracking powered by Google Gemini AI synthesizes data from multiple cricket APIs. Our ICC Rankings dashboard fetches real-time player standings through RapidAPI integration with 1-hour server-side caching."

**[20-35s] Advanced Analytics**
> "The Win Probability Calculator uses gradient boosted trees trained on BigQuery datasets. Player Form Dashboard leverages 15 different metrics - strike rates, consistency scores, venue matchups - all computed server-side with SWR client-side caching for instant updates."

**[35-45s] Smart Features**
> "Form-based XI Recommendations analyze recent performance trends across formats. Player images dynamically load from Wikipedia API with ESPN Cricinfo CDN fallback, featuring a 5-tier graceful degradation system."

**[45-55s] Architecture**
> "Built on Vercel Edge Runtime with PostgreSQL via Prisma ORM. BigQuery integration for data warehousing. Framer Motion animations at 60fps. Fully responsive Tailwind CSS design."

**[55-60s] Closing**
> "CricMetrics - where Google Cloud, NVIDIA compute, and cricket data science converge. Built for submission to the Google Cloud AI Hackathon 2026."
